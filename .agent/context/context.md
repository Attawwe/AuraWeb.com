## Analyse Concurrentielle (Secteur Yonne / 89)

### Concurrent 1 : Au Service du Web — auserviceduweb.fr
- **Mots-clés ciblés** : création site internet Yonne Joigny Sens
- **Forces** : Fort accent sur WordPress, le SEO local et la formation individuelle. Bonne présence locale.
- **Faiblesses** : Positionnement très "freelance" qui peut manquer de l'aspect premium/agence que NewEra propose.
- **Design & UX** : Axé sur le fonctionnel WordPress classique.
- **Stack technique** : WordPress
- **À retenir** : Insister sur l'aspect "sur-mesure total" vs "thème préconçu".

### Concurrent 2 : Subotaï — subotai.fr
- **Mots-clés ciblés** : agence web Sens Yonne création site internet
- **Forces** : Offre assez complète (SEO, Google Ads, RS, formations). 
- **Faiblesses** : Positionnement à la fois agence et formation, peut diluer le message "expert création sur-mesure".
- **Design & UX** : Professionnel.
- **Stack technique** : WordPress / CMS
- **À retenir** : Mettre en avant la performance technique brute (React/Vite) face aux CMS traditionnels.

### Concurrent 3 : Visicod Communication — visicod.com
- **Mots-clés ciblés** : agence communication web Auxerre création site
- **Forces** : Ancienneté sur le marché (depuis 2008), expérience prouvée.
- **Faiblesses** : L'image peut parfois vieillir par rapport aux standards très modernes actuels.
- **Design & UX** : Standard d'agence classique.
- **Stack technique** : Varie selon projets
- **À retenir** : L'opportunité est de frapper fort avec le design "Dark/Premium" de NewEra pour donner un effet "Wahou" immédiat face aux acteurs historiques.

### Opportunités identifiées
- **Design Premium "Dark Mode"** : Très peu de concurrents locaux ont ce niveau de finition visuelle. C'est un énorme avantage concurrentiel.
- **Stack Technique Moderne** : React/Vite/Tailwind offre des performances largement supérieures aux agences locales souvent cantonnées à WordPress basique.
- **Approche "Pas de modèles préconçus"** : Argument fort pour contrer l'industrialisation du web dans la région.
